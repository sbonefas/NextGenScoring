Teams and Rosters

View a list of teams in the system database (Team Code, Active Roster, Home Stadium, Head Coach, and Assistant Coach). Edit, Delete, or Add a new team to the database

Gametime Scoring

User views list of games played this season by UW Basketball and selects/creates one. Select a game or create a new one. Then score the game in real-time and see current stats of game.

Options

Change the file path, edit and view the games directory, change autosave settings, set the home team side for scoring, and displayed specialty stats.
